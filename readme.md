PSO2es의 한국어 패치 프로젝트입니다.<br>
번역 방식 간단 설명<br>
{<br>
		"assign": "601301",<br>
		"jp_text": "번역 이전 일본어 원문",<br>
		"tr_text": "번역 이후 한국어"<br>
},<br>
  <br>
